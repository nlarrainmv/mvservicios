Bright Pattern

Protegemos sus datos

A través de FIREBOX WATCHGUARD protegemos su información controlando spyware, virus, aplicaciones maliciosas y redes de robots.

seguridad de acceso

  • ✓
    Firewall: Inspección de paquetes con control de estado y firewall de proxy.
  • ✓
    Proxies: HTTP, HTTPS, FTP, DNS, TCP/UDP, POP3, POP3S SMTP, IMAPS y Explicit Proxy.
  • ✓
    Protección contra ataques DoS, paquetes fragmentados y malformados, y amenazas mixtas.
  • ✓
  • VoIP: H.323, SIP, establecimiento de llamada y seguridad de sesión.
  • ✓
    Opciones de filtrado: búsqueda segura en navegadores.

VPN y Autentificación

  • ✓
    Cifrado: AES 256-128 bits, 3DES, DES.
  • ✓
    IPSec: SHA-2, IKEv1/v2, clave previamente compartida IKE, certificados de terceros, Suite B.
  • ✓
    Inicio de Sesión Único (SSO): Windows, macOS, sistemas operativos móviles, RADIUS, SAML 2.0.
  • ✓
    Autenticación: RADIUS, LDAP, Windows Active Directory, VASCO, RSA SecurID y MFA.

Administración de Servidores

  • ✓
    Inicio de sesión y notificaciones: WatchGuard, Syslog, SNMP versión 2/versión 3.
  • ✓
    Interfaces de usuario: Consola centralizada (WSM), Web UI, CLI para secuencia de comandos.

Certificaciones
